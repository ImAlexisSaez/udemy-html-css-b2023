Positioning

Los valores para la propiedad position son:

  • static
  • relative
  • absolute
  • fixed
  • sticky

Además, las podemos combinar con los valores top, right, bottom, left y z-index.

Static

Es el valor por defecto para position. El elemento encaja en el flujo normal del documento. Los parámetros top, right, bottom, left y z-index no tienen efecto.

Absolute

El elemento se quita del flujo normal del documento y ahora es relativo al elemento padre con la posición declarada de manera explícita.

Si no hay padre así, será relativo a la raíz.

No se deja espacio ocupado donde debería haber estado.

Pueden emplearse las propiedades top, right, bottom, left y z-index.

Fixed

El elemento se elimina del flujo normal del documento y se posiciona con respecto al contenedor raíz que posea.

No se deja espacio reservado donde debería haber ido.

Se pueden emplear las propiedades top, right, bottom, left y z-indez.

Cuando hagamos scroll en la web, el elemento nos acompañará en esa posición fija que hayamos declarado.
